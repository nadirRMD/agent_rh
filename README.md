# Agent RH

Minimal FastAPI app for the Agent RH project.

## Requirements

- Python 3.14 or newer
- `uv` recommended for dependency management

## Install

```bash
uv sync
```

If you prefer `pip`, you can also install the project directly:

```bash
pip install .
```

## Run

```bash
uv run agent-rh
```

The API will start on `http://127.0.0.1:8000` by default.
